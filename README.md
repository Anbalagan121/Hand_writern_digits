# Handwritten Digit Recognition - Streamlit App

This is a production-ready AI web application that recognizes handwritten digits using a Convolutional Neural Network (CNN) trained on the MNIST dataset.

## Features

- **Home**: Project overview and workflow.
- **Digit Prediction**: Upload an image of a digit and get an instant AI prediction.
- **Draw Digit**: Use the interactive canvas to draw a digit and see real-time inference.
- **Dataset Explorer**: Explore the dataset distribution and sample images.
- **Model Performance**: Visualize the model's accuracy, loss, and confusion matrix interactively.
- **Model Insights**: Understand the CNN architecture and its workings.
- **About**: Developer information.

## How to Run Locally

### Prerequisites

Ensure you have Python installed. You can install all dependencies via:

```bash
pip install -r requirements.txt
```

### Starting the Dashboard

Run the following command from the project root:

```bash
streamlit run app.py
```

This will start the local server and open the web dashboard in your default browser.
